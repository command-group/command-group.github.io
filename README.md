## 简介

**男性主义**（英语：masculism），又作**男权主义**，是指一个主要以男性经验为来源与动机的社会理论与政治运动。其目标是**消除对男性的歧视，或提倡和强调被视为典型男性特质的价值观**。同时，这些术语也可能涉及男性权利运动或男性运动，甚至包括某种形式的**反女性主义立场**。
